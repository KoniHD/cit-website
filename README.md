This repo should serve as a template to create a short profile page for CIT students at TUM

## Motivation

Other universities such as CMU and Stanford already make broad use of institutional profile pages to improve visibility (for example: [CMU](https://computing.cs.cmu.edu/help-support/web-publishing), [Stanford](https://profiles.stanford.edu/)).  
Therefore this template is meant to make profile publishing easier for CIT students and to encourage wider adoption of personal academic webpages.

# Usage

1. Fill in your personal information in [`'profile.yml'`](profile.yml).
2. Replace [`'photo.webp'`](photo.webp) with your own image.
3. *Optional:* Add SEO meta tags for site ownership verification to [`'index.html'`](index.html)
3. Follow the instructions on the [wiki](https://wiki.ito.cit.tum.de/bin/view/CIT/ITO/Docs/Guides/Helpdesk/EigeneHomepage/) to make your site public.
